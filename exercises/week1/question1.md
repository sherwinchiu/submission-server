## 1.1 Hello World

Print out **"Hello World!"** into console using the basic template for C++ and std::cout.

### Example

Expected Output:

```
Hello World!
```

### Hints

Hint 1: Use std::cout <<

Hint 2: Use quotation marks around "Hello World!"

Hint 3: The line of code to write is std::cout << "Hello World!";
